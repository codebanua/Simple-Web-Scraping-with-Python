Git. 

## Installation

```sh
$ pip install requests
$ pip install bs4
$ pip install html5lib
$ pip install gunicorn
$ pip install Flask   
$ pip install Flask-Assets    
$ pip install Flask-Babel
$ pip install Flask-Script
```
